# Dataset of wire and blade alignment cross-correlation (ccf)

This dataset is the alignment cross-correlation of `x3p` wire and blade
cuts.

## Usage

``` r
cors_df
```

## Format

A tibble objects with 3 variables:

- lag: The lag of the alignment position.

- ccf: The cross-correlation of the aligned position.

- ns: The number of observations the ccf is based on.
